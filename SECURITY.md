# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability within this project, please send an email to bniladridas@gmail.com. All security vulnerabilities will be promptly addressed.

## Supported Versions
The latest two versions of the project are supported. Older versions may not receive security updates.

## Security Measures
- No personal data is collected.
- All data is processed in real-time without storage.
- Regular security audits are conducted.

## Additional Resources
For more information on security best practices, refer to the [OWASP Top Ten](https://owasp.org/www-project-top-ten/).
